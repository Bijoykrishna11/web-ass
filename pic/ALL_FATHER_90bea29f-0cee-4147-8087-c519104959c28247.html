Question:

Program Plan Intro

To design the circuit to make insertion tool active or inactive by proper positioning of PCB and the component to be inserted must be in chamber. Each of these conditions are indicated by a HIGH voltage whereas the insertion tool requires the LOW voltage to get activated.

Explanation of Solution

On the basis of the given information, there are three variables:

  1. PC board’s position (whether proper or improper)
  2. Component in the Chamber (whether present or absent)
  3. State of the insertion tool (whether active or inactive).

Here, the state of PC board’s position and Component in the chamber determine the output state of the insertion tool to be active or inactive. So, the first two variables will be input variable whereas the third one i.e. state of the insertion tool will be output variable.

Let us draw the table for all the cases possible with above three variables-

 Input

 Output

State of the insertion tool
Not proper Absent Inactive
Not proper Present Inactive
Proper Absent Inactive
Proper Present Active

It is given that the intrusion tool is active when a LOW voltage is present.

That means the following terms denote following type of Voltages-

“Active”=LOW, “Inactive”= HIGH.

“Not proper” =LOW and “Proper” = HIGH.

And

“Present” = HIGH and therefore “Absent” =LOW.

Let us draw the truth table on the basis of the above description-
